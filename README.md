# 🔗 URL Shortener with Authentication & History

A full-stack URL Shortener application that allows users to create short links, manage them securely with authentication, and track their URL history.

---

## 🚀 Features

### 🔐 User Authentication
- Register & Login functionality  
- Secure password handling  
- Session/JWT-based authentication  

### ✂️ URL Shortening
- Generate short URLs using **nanoid**  
- 8-character unique ID for each URL  

### 📜 User History
- Track all URLs created by the user  
- View original and shortened links  

### 🔁 Redirection
- Automatically redirect short URL to original URL  

### 🧾 Clean UI
- Simple and user-friendly interface  

---

## 🛠️ Tech Stack

### Frontend
- React.js  
- Tailwind CSS  

### Backend
- Node.js  
- Express.js  

### Database
- MongoDB  

### Other Packages
- nanoid (for generating short unique IDs)  
- bcrypt (for password hashing)  
- jsonwebtoken (for authentication)
